# 📚 Sistema de Planos de Aula - Academia Nacional de Polícia

Sistema web para criação e gerenciamento de planos de aula dos cursos da Academia Nacional de Polícia Federal.

## 🚀 Acesso Direto

**🌐 [Clique aqui para usar o sistema](https://wellpib.github.io/planos_aulas/sistema_planos.html)**

---

## 📋 Funcionalidades

### ✅ **Gerenciamento de Cursos:**
- **Agente de Polícia Federal** (4 aulas)
- **Escrivão de Polícia Federal** (20 aulas) 
- **Delegado de Polícia Federal** (10 aulas)
- **Papiloscopista Policial Federal** (4 aulas)
- **Perito Criminal Federal** (3 aulas)

### 🎯 **Recursos Principais:**
- ✅ **Editor de planos** individual por aula
- ✅ **Planos de disciplina** completos
- ✅ **Geração automática** de documentos .docx
- ✅ **Sistema de backup** integrado
- ✅ **Salvamento automático** no navegador
- ✅ **Controle de progresso** visual
- ✅ **Interface responsiva** e intuitiva

### 📄 **Campos por Plano de Aula:**
- Dados gerais (curso, disciplina, avaliação, duração)
- Objetivos específicos
- Conteúdos programáticos
- Estratégias de ensino (cronograma detalhado)
- Recursos materiais e didáticos

### 📑 **Geração de Documentos:**
- **Formato .docx** (online) - Documento profissional formatado
- **Formato .txt** (offline) - Alternativa para uso local
- **Templates oficiais** da Academia Nacional de Polícia

---

## 🖥️ Como Usar

### **💻 Uso Online (Recomendado):**
1. **Acesse**: [https://wellpib.github.io/planos_aulas/sistema_planos.html](https://wellpib.github.io/planos_aulas/sistema_planos.html)
2. **Navegue** pelos cursos na barra lateral
3. **Clique** em uma aula para editar
4. **Preencha** os campos necessários
5. **Gere** documentos .docx diretamente

### **📱 Uso Local:**
1. **Download** do projeto
2. **Execute** `iniciar_servidor.bat` (Windows)
3. **Acesse** `http://localhost:8000/sistema_planos.html`

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** + **CSS3** + **JavaScript ES6**
- **docx.js** - Geração de documentos Word
- **FileSaver.js** - Download de arquivos
- **localStorage** - Persistência local de dados
- **GitHub Pages** - Hospedagem gratuita

---

## ⌨️ Atalhos de Teclado

- **Ctrl + S**: Salvar aula/disciplina
- **Ctrl + G**: Gerar documento
- **Ctrl + B**: Fazer backup
